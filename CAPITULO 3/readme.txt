Para instalar SETUPTOOLS en windows:
===========================================
1. Inicie Powershell en windows 
2. Para instalar en python 2.7
	(Invoke-WebRequest https://bootstrap.pypa.io/ez_setup.py).Content | python -
   Para instalar en python 3.3
        (Invoke-WebRequest https://bootstrap.pypa.io/ez_setup.py).Content | py -3 -

Para instalar paquetes
===========================================
1. Ir a la ruta de easy_install

c:\python27\scripts> easy_install feedparser
